# 网页过滤器

用于抽取网页中你需要的信息，然后保存到你喜欢的任何地方。

### 使用方法

```bash
# 安装依赖
yarn install
# 开发
yarn start
# 构建 dll
yarn build-dll
# 构建
yarn build
```

### 本项目使用

- [ant.design](https://ant.design/)
- [webpack 2.0](http://webpack.js.org/)
